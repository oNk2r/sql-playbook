Data Types
- INT : Integer type (whole numbers)
- VARCHAR(100): varible length string up to 100 characters
- ENUM  : A string object with value chosen from list of permitted values eg. gender ENUM("Male","Female")
- DATE: stores date values e.g data_of_birth DATE
- TIMESTAMP: Stores data and time automatically
- BOOLEAN : stores TRUE or FALSE values often used for flags like is_active.

Constraints
- AUTO_INCREMENT : Automatically generates a unique number for each row
- PRIMARY KEY : uniquely identifies each row in table [can't be duplicate ]
- NOT NULL : no null values
- UNIQUE : all values in column are different
- DEFAULT : set default value if not provided
