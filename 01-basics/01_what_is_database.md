A database is a way to store data in a format which makes it easily accessible.

Query - A command to a database , by following the syntax

CRUD operations to be performed in database

DBMS - Database Management System
we interact database through DBMS , Query written to DBMS and DBMS does opertion for you in database.

e.g postgreSQL , MySQL , MS-SQL

Relational DBMS 
- Fixed and predefined schema
- Data is structured tables(rows and columns).
- SQL(structured Query Language) for defining and manipulating data.
- Tables can be linked using foreign key
e.g postgreSQL , MySQL , MS-SQL

Non-Relational DBMS 
- no fixed scheme , flexible data storage and modeling.
- SQL is not mandatory for data manipulation and querying
- mostly for Read operations.
e.g MongoDB , redis , neo4j , cassandra
